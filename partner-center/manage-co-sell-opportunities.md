# <a name="manage-co-sell-opportunities-in-partner-center-to-define-deals-and-better-meet-customer-needs"></a>管理合作伙伴中心的共同销售机会，以定义交易并更好地满足客户需求

**适用于**

- 合作伙伴中心
- 引荐
- 与 Microsoft 联合销售
- 交易共享

**相应的角色**

- MPN 合作伙伴管理员
- 引荐管理员

您可以利用共同销售机会。  共同销售机会是指由多个参与方完成的协作式销售，以满足客户需求的机会。 本部分将帮助你了解如何在合作伙伴中心利用共同销售的强大功能来有效地共同销售并收购新客户。

## <a name="types-of-co-sell-opportunities"></a>共同销售机会的类型

**共同销售**-这是一个或多个 Microsoft 销售代表积极参与的交易，以帮助解决客户问题。 商机可能源自一个客户帐户，其中，Microsoft 销售代表邀请合作伙伴参与共同销售活动，或从合作伙伴那里获得需要其他 Microsoft 销售帮助来结束交易的合作伙伴。

**专用管道交易**–当合作伙伴决定独立于合作伙伴中心创建的交易时，就会发生这种情况。 合作伙伴可以选择在创建交易后邀请 Microsoft 卖方，使其成为合作销售活动。 此交易的详细信息不会与 Microsoft 销售团队共享，直到合作伙伴明确寻找 Microsoft 销售帮助。

> [!NOTE]
> 尽管专用管道的处理并不是真正的协作性，但在这里，我们提供了一个简单**途径，可**用于将任何渠道的任何渠道升级到共同销售的机会。

## <a name="navigating-the-co-sell-section"></a>导航共同销售部分

共同销售部分中有三个顶级选项卡。

**入站**–此选项卡中提供了从 Microsoft 卖方向合作伙伴发送的用于协作销售的所有机会。

**出站**–此选项卡中提供合作伙伴创建的所有商机。合作伙伴可创建两种不同类别的机会-共同销售和专用管道。 

**收藏夹**–入站或出站销售机会选项卡中标记为收藏的机会将显示在此选项卡中。通过单击列表视图中机会上的心形图标，将任何机会标记为收藏。 只需再次单击相同的图标即可从收藏夹中删除此机会。


## <a name="creating-a-co-sell-opportunity"></a>创建共同销售机会 

> [!IMPORTANT]
> 若要与 Microsoft 共同销售，合作伙伴需要一个商业就绪的解决方案、一个共同的销售团队和一个强大的走向市场策略。 它们还必须满足特定的附加条件。 有关详细信息，请参阅[成为共同销售就绪](https://partner.microsoft.com/reach-customers/selling-with-microsoft#become-ready)。


您可以创建两种类型的共同销售机会。 

### <a name="select-your-customer"></a>选择客户

选择此特定交易的客户。 输入其位置，然后键入名称。 自动建议会根据搜索条件开始显示建议。 由于许多公司在同一位置有多个业务实体，你可能会看到多个结果。 如果不确定选择哪一种，可以确认哪个 D-N-S 号码属于您的客户。 请确保为你使用的客户选择完全匹配项。

单击 **"选择客户**" 后，系统会提示输入联系人的姓名、电话号码和电子邮件地址，作为此交易的主要联系点。 这些详细信息是可选的，仅当你打算邀请 Microsoft 销售人员帮助你参与服务时，这些详细信息才是必需的。 选择“**下一页**”。

> [!IMPORTANT]
> 请确保已获得客户同意向 Microsoft 提供其联系信息，以应对交易。 Microsoft 可能会使用此信息直接与客户联系。

### <a name="deal-details"></a>交易详细信息

输入当前交易的详细信息。 下面定义的字段可能会发生变化，因为你正在积极地与客户合作来结束交易。

| **字段名称** | **必需/可选** | **详细信息** |
|-------------|--------|-------|
|**交易名称** | 必需 | 用于在以后标识交易的友好名称。 |
|**估计值** | 必需 | 基于创建交易时可用的信息的交易价值。|
|**估计结束日期**| 必需| 您希望结束客户交易的截止日期。 |
|**CRM ID**| 可选 | 用各自 CRM 中的机会 ID 标记处理跟踪目的。|
|**市场营销活动 ID**| 可选 | 捕获导致交易的营销活动。 如果你标记了来自具有相同 ID 的市场活动的所有交易，则这一归档可帮助你跟踪某一市场活动的投资回报。|
|**说明**| 可选 | 更新所有最新信息，为公司的其他员工提供对同一交易的其他员工的可见性，或者尝试了解该交易的当前状态。 你还可以使用它作为记录的通信，以便与你的公司在 Microsoft 卖方/其他合作伙伴之间进行讨论。|

### <a name="add-your-employees"></a>添加员工

添加完交易详细信息后，添加将处理此特定交易的员工。 需要输入员工的姓名、电话号码和电子邮件地址。 这些详细信息是必需的，并且你需要至少一个联系人，其中包含为你输入的所有详细信息来创建交易。 即使在创建交易后，也可以更改这些详细信息。 你先前交易中的最近的联系人将显示在右侧，使你能够快速将它们添加到交易中。

### <a name="add-solutions"></a>添加解决方案

在本部分中，你需要提供与将作为此交易一部分的解决方案相关的信息。 这是必需部分，其中必须添加至少一个解决方案才能创建交易。 可以在创建交易后更改解决方案的详细信息。 有多种类型的解决方案可以添加到交易中，如下所述

- **我公司的解决方案：** 这些是公司发布的共同销售就绪解决方案
- **Microsoft：** 这些是 Microsoft 拥有的解决方案
- **其他第三方解决方案：** 这些是由 Microsoft 共同销售生态系统中的其他合作伙伴发布的共同销售就绪解决方案

提供解决方案信息后，选择 "下一步" 以转到可在其中确定销售类型的部分。 可以使用两个选项：

**专用管道交易**：如果在此步骤中不邀请 Microsoft 并创建订婚，则其类型为私有管道。 Microsoft 卖方无法查看此交易的详细信息。

**共同销售交易：** 如果将 **"是否喜欢 Microsoft 的帮助？" 这**一问题切换为 "是"，交易就会成为一种共同销售的交易，Microsoft 卖方可能会帮助你结束交易。 Microsoft 的帮助请求不能保证 Microsoft 卖方会参与此交易。 Microsoft 销售代表有14天的时间决定他们是否要参与。 在 "注释" 部分中，请确保确定所需的帮助类型。

## <a name="responding-to-a-co-sell-opportunity"></a>响应共同销售机会

每个机会将经历自己的生命周期。

### <a name="received-stage"></a>已接收阶段

在此阶段，如果你已收到来自 Microsoft 卖方或 Microsoft 共同销售生态系统中其他合作伙伴的新联销售机会，请查看详细信息，如果你想要详细了解其业务需求，可随时联系客户。 您可以在此阶段执行两项操作。 接受或拒绝引用：

- **接受：** 输入交易的名称，编辑预计的交易价值，并根据您的评论进行估计的购买时间范围。 建立与客户的联系后，应在 "**说明**" 字段中提供信息，以详细了解客户所需的信息。 你可以根据需要在此处输入你的 CRM ID （仅适用于你的参考），这是一个市场营销活动 ID，该 ID 导致了相应的机会，并添加了公司中将要处理此交易的联系人。 

- 完成后，选择“下一步”****。 我们会将引用转移到**下一阶段**，这意味着你计划主动与客户合作以满足其需求。 我们还将使用此信息来帮助你在将来查找类似的交易。

- **拒绝**：选择要拒绝交易的原因并添加要包括的任何便笺，然后选择 "**关闭交易**"。 我们会将其存档为 "已**拒绝**"，并通知 Microsoft 或合作伙伴向你发送此机会。

- 如果在分配的时间内未做出响应（目前为14天），我们会将其存档为**过期**，并通知 Microsoft 或向你发送此机会的合作伙伴。


### <a name="accepted-stage"></a>接受阶段

努力与客户达成交易。 如果要更改为已接受的引用提供的任何信息，请选择 "**编辑**"。 然后，你可以更新交易名称、估计购买日期、估计值、注释、CRM ID 和/或营销活动 ID。  你还可以选择 "**添加员工**" 以提供正在处理交易的任何其他人员的姓名、电话号码和电子邮件地址。 还可以根据客户需求编辑解决方案。

默认情况下，已创建的所有交易都处于接受阶段。

完成后，可以执行两个操作之一，将交易标记为 "**赢单**" 或 "**丢失**" 以报告结果，以便我们可以相应地对其进行存档。

> [!TIP]
> 对于某些合格的解决方案，选择 "赢单" 后，系统将要求你提供其他信息来注册交易。 Microsoft 将审查你在此处提供的信息，并且在审查过程中可能会要求你提供其他详细信息。 有关详细信息，请参阅注册交易。

### <a name="archived-stage"></a>存档阶段

这是一个终端阶段，其中的所有机会最终都会达到。 你可以在此阶段查看**赢、丢失、拒绝**和**过期**的所有潜在顾客。 此阶段中没有可执行的操作。

## <a name="getting-more-co-sell-opportunities"></a>获得更多共同销售机会

下面是一些提示，可帮助你获得适用于你的业务的更多共同销售机会：

- **快速响应交易**。 当你及时响应传入请求时，我们将在将来的合作伙伴搜索结果中逐渐提高你的可见性。 确保你的团队按照你的意图快速作出回应。
- **审慎选择接受的交易**。 我们会监视你接受和拒绝的交易类型，并使用此信息帮助你查找类似的交易。 接受不太适合的交易不会改善搜索结果，可能会影响收到的机会的质量。
- **报告预计的交易量、关闭日期和交易的最终状态**（赢单或丢单）。 我们将使用此信息继续提供质量检索。
