# <a name="pricing-and-offers-for-office-365-dynamics-crm-enterprise-mobility-suite-azure-and-more"></a>Office 365、Dynamics CRM、企业移动性套件、Azure 等的定价和优惠


**适当的角色**：全局管理员 |用户管理管理员 |管理代理 |MPN 合作伙伴管理员 |销售代理 |计费管理员

若要查看最新的云解决方案提供商计划和产品/服务，请在合作伙伴门户仪表板中，转到“销售”>“定价和产品/服务”。 对于不同类型的可用产品，你将找到单独的价目表。 可在“定价和产品/服务”页面上查看以下价目表：

- **基于许可证的服务** 包括 Office 365、企业移动性和安全 E3 以及 Dynamics 365 的定价信息。 基于许可证的定价部分包括当前和预览版定价以及产品/服务列表矩阵。 价目表包括标价和估计零售价格 (ERP) 提供所有支持的货币。 产品/服务列表包括市场可用性和有关产品/服务的其他重要信息。 每个月的第一天将更新这些文件。
- **基于使用情况的服务** 包括 Microsoft Azure 和 Visual Studio 的定价信息。 还可以 [在 CSP 定价计算器中使用 Azure 服务](https://azure.microsoft.com/pricing/calculator/)。 基于使用情况的下载同时包括所有支持的货币的 CSP 价格列表，以及 ERP 中的共享服务定价文件。 每个月的第一天将更新这些文件。
- **Microsoft Azure 保留实例** 包括 Azure 预订实例的所有支持货币的定价信息。 定价下载还包括 ERP 中的共享服务定价。 每个月的第一天将更新这些文件。
- 对于所有受支持的货币，**软件订阅** 都包含基于字词的软件订阅的定价。 此价格文件包括标价和 MSRP 的所有支持货币。 每个月的第一天将更新这些文件。
- **Azure 计划定价** 包括 azure 计划使用服务和 azure 计划保留定价的定价信息。 价格是直接价格或 ERP，可以为任何给定的受支持市场检索。 这些文件的数据每天都会刷新。
- **使用汇率** 计算美元与合作伙伴本地货币之间的计费。 费率在每月的第一天更新，在给定月份保持为 true，该费率可用。 这些文件在每月的第一天更新。
- **市场** 包括 Microsoft 商业市场中 ISV 解决方案的定价。 按市场检索价格。 这些文件的数据每天都会刷新。

> [!Note] 
> 只有具有交易能力的 CSP 合作伙伴才能查看和下载价目表。 间接经销商应 [联系其 CSP 提供商]( https://partner.microsoft.com/cloud-solution-provider/find-a-provider) ，请求有关定价的详细信息

## <a name="price-changes"></a>价格更改

价格变化是一种常见情况。 合作伙伴可以通过查看价目表预览来预见基于许可证的产品/服务的价格变化。 在合作伙伴中心仪表板上，打开 "定价和优惠" 页，查看价目表预览。

但是，基于 Azure 使用情况的定价没有预览。 合作伙伴可通过使用 RateCard API 来与 Azure 消耗价格变化保持联系，后者返回了那一天的计量价格。

|**产品类型**   |**产品示例**  |**预览可用** |**更改详细信息**|
|-----------------------|:-----------------------|:-------------------|------------------|
|基于许可证|Office，Dynamics，Intune，Windows Enterprise|30天|列出预览版价格列表中标记为 CHNG 的价格更改|
|基于使用情况|Azure 资源|不可用|更改上个月价目表的 " **更改历史记录** " 选项卡中可用的日志|
|软件||不可用|按月手动比较价目表|
|预留|虚拟机，预付费|不可用|按月手动比较价目表|

基于使用情况的价格在一个月内可能会发生变化。 若要获取这些 Azure 资源的"当前"每日定价，合作伙伴需要调用 RateCard API。 

> [!Note] 
> 订阅价格更改仅适用于续订期间。 合作伙伴的每月费用是根据购买价格或创建订阅时的价格确定的。 如果在获取年度期限后价格增加或降低，则合作伙伴在续订之前不会收取更改的价格-通常在 12 个月期限。

|**Segment**   |**谁需要符合条件**   |**合作伙伴符合客户资格**|**已启用的产品类型**|

## <a name="add-on-offer-types"></a>附加产品/服务类型

基于许可证的服务可以作为基本产品/服务或加载项获得。 仅可通过合作伙伴中心目录发现和可购买基本产品/服务。 在购买基本产品/服务后，必须应用外接程序。 基于许可证的价目表 " **辅助许可证类型** " 列包含有关每个产品/服务及其类型的信息。 基本产品/服务的价格列表辅助许可证类型列中的值 **不特定** ，可以在目录中购买。 无法在目录中购买 **外接程序** 的辅助许可证类型值。 购买以下外接程序：

1. 请参阅产品/服务列表矩阵，查看在购买外接程序之前需要购买的产品/服务 Id 列表。
2. 从目录中购买基本产品/服务
3. 从 "客户" 列表中导航到你的客户。 单击刚刚购买的基本产品/服务的订阅。 在"管理订阅"页上，可以看到可应用于基本套餐的可用加载项。

> [!Note] 
> 某些基本产品/服务具有 **"附加产品** 许可证 **"的"单元类型"值**。 对于基本产品/服务，这只是意味着购买后不会分配用户许可证。 如果可以在目录中购买产品/服务，则无论用户界面中的单位类型如何，它都是基本产品/服务。

## <a name="pricing-between-azure-and-non-azure"></a>Azure 与非 Azure 之间的定价

定价因不同类型的产品/服务而不同。 基于许可证的定价通常是给定月份每个许可证的金额。 基于使用情况的定价取决于使用具有关联计量 ID 的给定资源。 合作伙伴不收取获取 Azure 订阅的费用。 但是，合作伙伴需要为 Azure 订阅下的不同部署消耗的资源付费。 基于使用情况的价目表中的定价围绕 Azure 中的不同资源计量器 ID 进行组织。

Azure 预留是特定资源类型（虚拟机）的基于期限的购买。 购买 Azure 预留可让合作伙伴预先 (一年或三年) 预留给定的虚拟机。 预留可节省合作伙伴资金，并确保其虚拟机在期限内始终可用。 合作伙伴可以按照基于使用情况的资源计量的 ID 来调整其预留。 计量 ID 在整个资源中是一致的，无论合作伙伴是购买虚拟机，还是只是将虚拟机部署为基于使用情况的资源。

## <a name="offers-matrix"></a>产品/服务矩阵

## <a name="offer-limits"></a>产品/服务限制

某些基于许可证的产品/服务具有禁止同一客户多次购买的规则和限制。 这些规则适用于大多数试验和许多小型企业产品/服务。 **小型企业产品/服务** 是由具有小于300的最大许可证计数的产品/服务定义的。

这些采购约束定义为产品/服务配置的一部分，可通过查看产品/服务列表矩阵来找到。 两列数据结合使用来定义强制：1。 提供限制范围和2。 提供限制。 在购买过程中强制实施约束。 合作伙伴中心的目录将禁止合作伙伴购买超过规则所允许的其他产品/服务。 任何违反约束的尝试都将导致错误。

产品/服务限制范围在产品/服务列表矩阵上记录为列，并且其值可以为 None、Lifetime 或并发。 

- 无需约束即可购买 **无** 限制的产品/服务。
- **生存期** 提供只能购买一次。
- 可以根据产品/服务的产品/服务 **限制** 值的允许次数多次购买 **并发** 产品/服务。 大多数试验的生存期提供限制范围为 "1"。 大多数小企业产品/服务具有一个提供限制为 "2" 的并发提供限制范围。

> [!IMPORTANT]
> 即使取消了产品/服务，也会强制实施并发限制。 必须完全取消产品/服务，然后取消预配，以释放额外的空间，允许进行其他购买。

## <a name="multi-year-term-offers"></a>多年份条款提供

### <a name="36-month-offers"></a>36月优惠

大约有为期三年的 50 Dynamics 产品/服务。 它们通过 **(36 mo)** 在产品/服务的标题中进行标识。 这些产品/服务与每年提供的产品/服务类似。 唯一的区别在于它们的术语。 这些产品/服务的有效期为三年，以便订阅在三年后自动续订（而不是一年）。 下面概述了这些功能的工作原理：

- 条款为36个月，订阅将在三年后自动续订
- 合作伙伴可以取消或更改整个订阅期限内的许可证数量
- 年度续订将以购买时间为三年的价格
- 计费频率仍为每年或每月

### <a name="72-month-offers"></a>72月优惠

Microsoft 365 A1 基础产品/服务有六年的期限。  在购买此基础产品/服务后，Office 365 A1 外接程序可用。 

|**产品/服务名称**   |**产品/服务 ID**   |类型|
|-------------------|-----------------------|----------------------------|
|Microsoft 365 A1|778a4dce-0014-4d53-8647-314ef2b091d2|基本优惠|
|适用于教职员的 Office 365 A1 (设备) |0757d14e-7c57-456f-8dab-47d164f2ff1f|加载项|
|Office 365 A1 学生版 (设备) |bae285a9-d56b-4384-b02f-38adc61a6f12|加载项|

下面总结了这些产品/服务如何工作：

- 期限为 72 个月或 6 年
- 订阅未续订，在六年后过期
- 产品/服务上的计费频率显示为年度，但在获取订阅后，合作伙伴会先在第一张发票上计费
- A1 72 个月套餐的订阅在购买后锁定，无法取消，并且首次购买后无法将许可证计数添加到订阅
- 此订阅不可取消且不可退款。
