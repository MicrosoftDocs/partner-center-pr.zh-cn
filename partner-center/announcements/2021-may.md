- 功能

## <a name="default-bank-profile-functionality-is-available-in-partner-center-for-incentive-enrollment"></a><a name="7"></a>可在合作伙伴中心使用默认银行档案功能进行奖励注册

### <a name="categories"></a>类别

- 日期：2021-05-07
- 功能

### <a name="impacted-audience"></a>影响受众

奖励管理员

### <a name="details"></a>详细信息

我们很高兴宣布在合作伙伴中心提供默认银行档案功能。 通过此功能，奖励管理员可设置默认银行档案并将其分配到 Microsoft 合作伙伴网络 (MPN) 位置。 设置好这些档案后，当合作伙伴想要就该 MPN 位置注册任何奖励计划时，Microsoft 将使用这些档案。

有关详细说明，请查看[设置默认银行档案](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles#set-up-a-default-bank-profile)。

#### <a name="enrolling-in-an-incentive-program-when-a-default-bank-profile-has-been-assigned"></a>在已分配了默认银行档案的情况下注册奖励计划

如果你尝试在奖励计划中注册一个 MPN 位置，而该位置已设置了默认档案，那么你在选择“注册”并接受邀请时，将看到该默认档案。 你还将看到你已为该 MPN 位置创建的任何可用税务档案。 如果 Microsoft 具有所有必需的银行和税务档案详细信息，系统将提示你选择“提交”来完成注册。

你还可选择与默认银行档案不同的银行档案。 如果 Microsoft 要求就付款或税务档案/货币提供其他详细信息，系统将提示你选择“继续”，并将你重定向到“付款和税务档案”页面来提供缺少的详细信息。

只有当你为注册提供的付款和税务档案已经过 Microsoft 的验证后，注册才被视为已完成。 请在[此处](https://docs.microsoft.com/partner-center/incentives-create-and-manage-your-payout-and-tax-profiles)了解有关付款和税务档案的详细信息。

### <a name="questions"></a>是否有任何问题?

如果执行这些步骤时需要帮助，请访问 Microsoft 合作伙伴网站的[支持](https://partner.microsoft.com/support/?stage=1)部分来查看你的支持选项。

## <a name="csp-perpetual-software-coming-soon-to-china"></a><a name="5"></a>即将向中国推出 CSP 永久性软件

### <a name="categories"></a>类别

- 日期：2021-05-05
- 功能

### <a name="summary"></a>摘要

在财年第一季度（2022 财年第 1 季度），将向 Microsoft China Corporation Limited (MCCL) CSP 计划推出永久性软件。

### <a name="impacted-audience"></a>影响受众

已参加 MCCL CSP 计划的合作伙伴

### <a name="details"></a>详细信息

CSP 计划的参与人数呈指数增长，已有超过 90,000 家公司加入。 这是我们针对中小型企业和更小型的采用新商业体验的公司客户而推出的主要合作伙伴销售计划。 合作伙伴可为客户创建和提供差异化产品/服务，从而除了他们自己的增值解决方案和服务以外，还可从 Microsoft 商业市场销售 Microsoft 云组合和第三方解决方案。 他们能够管理整个客户参与生命周期，还能以受信任的顾问的身份与客户建立信任关系。

#### <a name="expanding-opportunities-for-partners"></a>扩大合作伙伴的机会

在此推动下，Microsoft 将在 Microsoft 2022 财年第 1 季度发布不带软件保障 (SA) 的商业永久性软件许可证来便于通过 *MCCL CSP 进行本地部署，从而扩大 CSP 合作伙伴的机会。 这为客户提供了许可灵活性，让他们可以轻松实现数字化转型。

如[之前公布的](https://blogs.partner.microsoft.com/mpn/expanding-opportunities-for-partners-in-the-cloud-solution-provider-program/)，从 2022 年 1 月 1 日开始，不可再通过开放许可计划续订 SA 或购买新的软件许可证。 而是应通过 MCCL CSP 进行新的仅限许可证的购买。

*这仅适用于 MCCL CSP。 对世纪互联 CSP 没有任何影响。

### <a name="next-steps"></a>后续步骤

已注册或即将注册 MCCL CSP 的合作伙伴应查看以下资料，并与其组织中相应的利益干系人分享信息，来做好应对准备：

- [面向大中华地区 (GCR) 的 CSP 和合作伙伴中心准备资料](https://partner.microsoft.com/resources/collection/software-in-csp-cht#/)：介绍性概述、许可指南、合作伙伴中心订购事项、API 操作视频、常见问题解答等。 副标题以简体中文或繁体中文的形式提供。

- [Microsoft 许可资讯上的 Microsoft 开放许可计划变更通知](https://www.microsoft.com/zh-cn/licensing/news/microsoft_open_license_program_changes)：关于开放许可计划和许可选项变更的更多详细信息。 如需相关的开放许可计划变更准备资料，另请查看 [Microsoft 合作伙伴网站](https://partner.microsoft.com/resources/collection/csp-open-evolution-to-a-better-experience#/)。

## <a name="corrections-to-may-1-2021-license-based-services-price-list"></a><a name="3"></a>对 2021 年 5 月 1 日的基于许可的服务价目表的更正

### <a name="categories"></a>类别

- 日期：2021-05-04

- 产品/市场

### <a name="impacted-audience"></a>影响受众

在云解决方案提供商计划中交易基于许可证的服务的间接提供商和直接帐单合作伙伴。

### <a name="details"></a>详细信息

在 2021 年 5 月 1 日就基于许可证的服务发布的价目表中，Microsoft 365 商业标准版的美元价格不正确。 发布的标价是 8 美元，实际应该是 10 美元。 2021 年 5 月 3 日上午 11:30（太平洋标准时间）更新了基于许可证的服务的价目表，更正了这一错误。 这些更正仅适用于：

- 产品/服务名称：Microsoft 365 商业标准版

- 产品/服务 ID：031c9e47-4802-4248-838e-778fb1d2cc05

- 货币：美元

- 更正前：标价：8 美元

- 更正后：标价：10 美元

这些更改仅适用于上述产品/服务。 其他产品/服务未做更正。 这只是价目表上的错误，实际价格无误，这将影响这期间的发票。

### <a name="next-steps-and-resources"></a>后续步骤和资源

- 交易基于许可证的服务的合作伙伴应下载最新的价目表。
