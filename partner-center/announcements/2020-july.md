## <a name="new-capabilities-added-to-the-insights-reports-in-partner-center"></a><a name="6"></a>合作伙伴中心的“见解”报表中添加了新功能 


### <a name="categories"></a>类别

- 日期:2020-07-16
- 功能
 
 
### <a name="impacted-audience"></a>影响受众

Microsoft 合作伙伴网络中的所有合作伙伴组织 

### <a name="summary"></a>摘要

合作伙伴中心的“见解”报表中添加了新功能，合作伙伴可根据业务需求在该报表中执行临时数据分析，并查看组织内部员工所接受培训的整体情况。  这扩充了[现有“见解”报表](https://docs.microsoft.com/partner-center/partner-center-insights)的稳健集合。

新功能概述：

- **原始数据导出**，支持客户详细信息、订阅详细信息、Microsoft Azure 使用情况、Office 365 使用情况、Dynamics 365 使用情况、企业移动性和安全性 (EMS) 使用情况、PowerBI 使用情况、培训和认证以及 Microsoft Learn 等方面的原始数据导出。 有关更多详细信息，请参阅[本文](https://docs.microsoft.com/partner-center/pci-download-reports)。

- **新的培训仪表板**，提供组织内部员工所接受培训的整体情况。 有关更多详细信息，请参阅[本文](https://docs.microsoft.com/partner-center/pci-training-dashboard)。

我们鼓励你探索这些新功能，并使用“见解”选项卡中提供的内嵌反馈工具分享你的反馈。

### <a name="what-is-insights"></a>“见解”仪表板的作用是什么？

通过合作伙伴中心的“见解”仪表板，你可以跨云产品（如 Office 365、Azure、Dynamics 365）和许可模式（如云解决方案提供商 (CSP) 和企业协议 (EA)）获取合作伙伴组织关键绩效指标 (KPI) 的 360 度统一视图。 还可获取智能见解，清楚地了解与 Microsoft 的业务，并满怀信心地做出数据驱动的决策。   

要查看此仪表板，请在合作伙伴中心的左侧导航菜单中选择“见解”。 此仪表板使用与当前 MyInsights 报表工具相同的基础数据，但提供了更丰富的视觉对象，以及更具洞察力的报表功能和 KPI，这些功能和 KPI 涵盖客户、订阅、许可证使用情况、胜任度和权益。 请经常返回查看，因为我们会继续在“见解”仪表板中引入新的报表和功能。

## <a name="perpetual-software-in-the-cloud-solution-provider-program"></a><a name="5"></a>云解决方案提供商计划中的永久软件 


### <a name="categories"></a>类别

- 日期:2020-07-15
- 产品/市场
- 状态：可用于一组符合条件的间接提供商及其间接经销商。 将于 2021 年 1 月正式发布
 
 
### <a name="impacted-audience"></a>影响受众

CSP 间接经销商以及符合条件的 CSP 间接提供商 

### <a name="details"></a>详细信息

在 Microsoft，我们致力于帮助每个组织完成数字化转型。 为此，我们不仅投资技术，还研究从获取技术到提供支持的端到端客户和合作伙伴历程。 通过这项工作，我们发现了实现自我转型的机会，即投资我们的商业平台。  我们还在不断简化许可计划，同时发展新的商业体验，以便改善为合作伙伴提供更多机会的方式。 

随着我们改进商业功能以便为客户和合作伙伴提供更好的购买体验，我们持续扩展产品组合，实现在云解决方案提供商 (CSP) 计划中提供永久软件许可证。 在 CSP 中添加软件许可证后，参与的合作伙伴便可以专注于向其客户提供各种解决方案和托管服务。 它还使客户能够过渡到云，并为合作伙伴提供混合云环境所需的灵活性。   

从 7 月 1 日开始，我们将通过一组符合条件的间接提供商及其间接经销商来增加此功能的采用，这些提供商和经销商将能够与 CSP 计划中的客户交易本地软件的许可证。 符合条件的合作伙伴由 Microsoft 管理，并且必须既是活跃的 CSP 间接提供商，又是开放许可计划的 Microsoft 授权分销商，该计划在当前支持在 CSP 中销售软件的地域运营。 从 2021 年 1 月开始，Microsoft 将进一步扩展此功能，使所有 CSP 直接账单合作伙伴、间接提供商和间接经销商都可以在 CSP 计划中销售软件许可证。    

### <a name="next-steps"></a>后续步骤

- 查看操作准备资源库集合中的所有准备材料：[云解决方案提供商计划中的永久软件](https://partner.microsoft.com/resources/collection/software-in-csp#/)。

## <a name="microsoft-has-ended-sales-of-the-microsoft-kaizala-pro-standalone-sku"></a><a name="2"></a>Microsoft 已终止销售 Microsoft Kaizala Pro 独立 SKU

### <a name="categories"></a>类别

- 日期:2020-07-01
- 键入：产品/市场
- 状态：现已推出

### <a name="summary"></a>摘要

Microsoft Kaizala Pro 不再向新客户或续订客户提供。 这些客户可以通过转变到其他 Microsoft 365 和 Office 365 许可证来访问 Kaizala。

### <a name="impacted-audience"></a>影响受众

销售 Microsoft Kaizala Pro 独立 SKU 的合作伙伴

### <a name="details"></a>详细信息

Microsoft 正将 Kaizala Pro 功能及其最常用功能的一部分添加到 Microsoft Teams 中，后者是 Microsoft 365 中用于团队合作的中心。

从今天（2020 年 7 月 1 日）开始，Microsoft 将终止向新客户或续订客户销售 Kaizala Pro 独立 SKU。 现有客户可继续通过其当前许可证访问该服务。 我们已于 2020 年 6 月 1 日向 Kaizala 客户公告了此项更改。

Kaizala 和 Teams 将继续作为下列 Microsoft 365 和 Office 365 SKU 的一部分供用户访问：

[Microsoft 365 商业版](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab%3aprimaryr2)，包括：  
- Microsoft 365 商业基本版

CSP Direct Bill Partner、Indirect Provider 和 Indirect Reseller

2021 财年 CSP 奖励计划将于 2020 年 10 月 1 日开始。 有关 2021 财年 CSP 奖励计划的详细信息将于 2020 年 8 月 1 日公开。
