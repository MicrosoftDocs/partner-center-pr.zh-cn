## <a name="new-microsoft-365-enterprise-voice-skus-for-teams"></a><a name="4"></a>适用于 Teams 的新的 Microsoft 365 企业语音 SKU

### <a name="categories"></a>类别

- 日期:2020-07-02
- 键入：产品/市场
- 状态：即将推出

### <a name="summary"></a>摘要

2020 年 8 月 1 日，Microsoft 将以新 Teams 产品/服务的形式发布新的 Microsoft 365 企业语音，其中包括电话系统、音频会议和国内通话套餐。

### <a name="impacted-audience"></a>影响受众

直接提供商、间接提供商、间接经销商 

### <a name="details"></a>详细信息

2020 年 8 月 1 日，Microsoft 将发布新的 Microsoft 365 企业语音 SKU。 Microsoft 365 企业语音是一种基于云的电话服务解决方案，允许用户在 Microsoft Teams 中进行 PSTN 呼叫和接收此类呼叫。 它包括电话系统、音频会议和国内通话套餐。

E5 将包含 120 分钟的国内主叫通话，因此将享受相当于企业语音套餐 1 的套餐。

目前，此新产品/服务将在除了美国和波多黎各之外的所有[通话套餐国家/地区](https://docs.microsoft.com/MicrosoftTeams/country-and-region-availability-for-audio-conferencing-and-calling-plans/country-and-region-availability-for-audio-conferencing-and-calling-plans)推出。 一旦获得有关此套餐在美国推出的更多信息，我们就会在此公告中详细说明。

**它是什么？**

企业语音是一种新的产品/服务，其中包括电话系统、音频会议和国内通话分钟数。 

**它的费用是多少？**

新产品/服务的价格在合作伙伴中心的“定价和套餐”页中的 8 月份价目表预览版中提供。
        
**它何时推出？**

这些产品/服务预期在 2020 年 8 月 1 日推出。 目前的 8 月份价目表预览版包含有关这些套餐的更多详细信息。  

**哪些客户可以购买它？**

- 商业客户、全球商业公共部门客户和非营利客户
- 不能供教育版客户使用。 尚未确定 GCC、GCC-High 和 DoD 的可用性

**如何购买它？**

企业协议 (EA)/企业订阅 (EAS)、CSP 或 Web Direct

**它在何处销售？**

所有当前的通话套餐销售国家/地区，**不包括美国和波多黎各**。

**先决条件是什么？**

M365 F3/E3 或 O365 F3/E1/E3

**在现有套餐的基础上增加了什么内容？**

M365/O365 E5/A5 将包含 120 分钟的国内主叫通话，因此将享受相当于套餐 1 的套餐。 增加的这项内容不额外收费，但客户需要使用新的“赠送通话分钟数”的 Microsoft 365/Office 365 E5/A5 SKU 才能享受通话分钟数。

## <a name="microsoft-has-ended-sales-of-the-microsoft-kaizala-pro-standalone-sku"></a><a name="2"></a>Microsoft 已终止销售 Microsoft Kaizala Pro 独立 SKU

### <a name="categories"></a>类别

- 日期:2020-07-01
- 键入：产品/市场
- 状态：现已推出

### <a name="summary"></a>摘要

Microsoft Kaizala Pro 不再向新客户或续订客户提供。 这些客户可以通过转变到其他 Microsoft 365 和 Office 365 许可证来访问 Kaizala。

### <a name="impacted-audience"></a>影响受众

销售 Microsoft Kaizala Pro 独立 SKU 的合作伙伴

### <a name="details"></a>详细信息

Microsoft 正将 Kaizala Pro 功能及其最常用功能的一部分添加到 Microsoft Teams 中，后者是 Microsoft 365 中用于团队合作的中心。

从今天（2020 年 7 月 1 日）开始，Microsoft 将终止向新客户或续订客户销售 Kaizala Pro 独立 SKU。 现有客户可继续通过其当前许可证访问该服务。 我们已于 2020 年 6 月 1 日向 Kaizala 客户公告了此项更改。

Kaizala 和 Teams 将继续作为下列 Microsoft 365 和 Office 365 SKU 的一部分供用户访问：

[Microsoft 365 商业版](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-products?&activetab=tab%3aprimaryr2)，包括：  
- Microsoft 365 商业基本版
- Microsoft 365 商业标准版
- Microsoft 365 商业高级版

[适用于一线员工的 Microsoft 365](https://partner.microsoft.com/resources/collection/Microsoft-365-firstline-offer-updates)，包括：
- Microsoft 365 F3（以前为 Microsoft 365 F1）和 Office 365 F3

[Microsoft 365 企业版](https://www.microsoft.com/microsoft-365/compare-microsoft-365-enterprise-plans)，包括： 
- Office 365 E1
- Microsoft 365 E3 和 Office 365 E3
- Microsoft 365 E5 和 Office 365 E5

[Microsoft 365 教育版](https://www.microsoft.com/education/buy-license/microsoft365)，包括： 
- Microsoft 365 A1 和 Office 365 A1
- Microsoft 365 A3 和 Office 365 A3
- Microsoft 365 A5 和 Office 365 A5

实现此更改是为了确保向新客户和续订客户授予许可，从而实现从 Kaizala 到 Teams 的转变。

## <a name="csp-incentive-programs-extended-through-fy21-q1"></a><a name="1"></a>CSP 奖励计划延长到 2021 财年第 1 季度
