## <a name="now-available-windows-7-year-2-esu-for-cloud-solution-provider-csp"></a><a name="6"></a>现已推出适用于云解决方案提供商 (CSP) 的 Windows 7 第二年 ESU

### <a name="categories"></a>类别

- 日期:2020-06-29
- 键入：产品/市场
- 状态：现已推出

### <a name="summary"></a>摘要

Windows 7 第 2 年扩展安全更新 (ESU) 定价现已输入[资源库](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)。

### <a name="impacted-audience"></a>影响受众

参与云解决方案提供商 (CSP) 计划的所有合作伙伴

### <a name="details"></a>详细信息

对 Windows 7 的支持已于 2020 年 1 月 14 日结束。 尽管许多客户正在部署 Windows 10，但 Microsoft 明白每个人都处于升级过程中的不同阶段。 Windows 7 第 1 年 ESU 的有效期将于 2021 年 1 月 12 日结束。 我们知道有的客户正在开始明年的计划。  

对于希望在 2021 年继续使用 Windows 7 设备的客户，我们现在将为云解决方案提供商 (CSP) 合作伙伴提供 Windows 7 第 2 年 ESU 的定价，以帮助客户决定是否使用 Windows 10。 有关详细信息，请访问[资源库](https://partner.microsoft.com/resources/collection/windows-esu-csp#/)。

## <a name="partner-center-relationship-removal-feature-update"></a><a name="5"></a>合作伙伴中心关系删除功能更新

### <a name="categories"></a>类别

- 日期:2020-06-29
- 键入：功能
- 状态：现已推出
- 
### <a name="summary"></a>摘要

从 2020 年 6 月 29 日起，云解决方案提供商 (CSP) Indirect Provider 和 Indirect Reseller 可以使用合作伙伴中心删除下游关系，不再需要提交合作伙伴中心支持票证。 这样可以在合作伙伴执行这些活动时节省大量时间。

### <a name="impacted-audience"></a>影响受众

云解决方案提供商 (CSP) 间接提供商和间接经销商

### <a name="scenarios-supported"></a>支持的方案：

- 与间接经销商建立关系的间接提供商可以使用合作伙伴中心删除该关系。
- 与最终客户建立关系的间接经销商可以使用合作伙伴中心删除该关系。

上述方案中的操作必须由合作伙伴中心管理员代理来执行。

### <a name="next-steps"></a>后续步骤：

- 有关详细信息，请访问[资源库](https://partner.microsoft.com/resources/collection/partner-center-relationship-removal#)。
- 查看操作准备情况库收集材料：合作伙伴中心关系删除功能更新。
- 在组织中共享此信息，以便相应团队可以查看和更新流程。

## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>计费频率默认从“按年”更改为“按月”

### <a name="categories"></a>类别

- 日期:2020-06-17
- 键入：功能
- 状态：即将推出

### <a name="summary"></a>摘要

在合作伙伴中心购买体验中，计费频率默认从“按年”更改为“按月”。

### <a name="impacted-audience"></a>影响受众

云解决方案提供商 (CSP) 直接提供商和间接提供商

### <a name="details"></a>详细信息

计费频率设置可确定针对合作伙伴已购买的服务向其计费的频率。 在合作伙伴中心目录中进行购买时，合作伙伴会在“审阅”步骤中看到此设置。 此设置默认为“按年”，因为已针对基于许可证的服务引入了按年计费。 根据合作伙伴的反馈，合作伙伴中心团队从 6 月 17 日起会将此默认值从按年更改为按月，以便更符合合作伙伴预期的情况 。 合作伙伴可以在审阅页上提交交易之前或在从管理订阅页提交交易之后对此设置进行更改。 请注意，仅支持按年计费的产品/服务才可继续沿用默认的“按年”计费。 此默认值更改仅适用于合作伙伴中心门户的用户体验，合作伙伴中心 API 的工作方式不会更改。

我们希望这种更改能降低合作伙伴在无意中意外购买按年计费订阅的可能性。

- 日期:2020-06-15
- 键入：产品/市场
- 状态：现已推出

### <a name="summary"></a>摘要

将通过新服务更新 Microsoft 365 F1。 将于 6 月 15 日发布新产品/服务以替换之前的 Microsoft 365 F1。

### <a name="impacted-audience"></a>影响受众

   |Microsoft 365 F1 试用版|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

希望购买新 F1 订阅的合作伙伴应在 6 月 15 日前使用以下产品/服务 ID。 合作伙伴可以在合作伙伴中心目录中查看这些 ID。 利用合作伙伴中心 API 的合作伙伴也可以使用以下信息。

具有当前 Microsoft 365 F1 产品的合作伙伴无需进行任何操作。 Exchange Kiosk 和 MS 搜索的服务计划将在 6 月底基于现有服务计划进行回填。

## <a name="new-and-end-of-life-microsoft-dynamics-365-and-power-platform-offers-and-products"></a><a name="1"></a>新的和已停用的 Microsoft Dynamics 365 和 Power Platform 产品及服务

- 日期:2020-06-11
- 键入：产品/市场
- 状态：即将推出

### <a name="summary"></a>摘要

Microsoft 将于 2020 年 7 月 1 日发布新的 Dynamics 365 和 Power Platform 产品及服务，并停用相应的反欺诈产品。

### <a name="impacted-audience"></a>影响受众

- Microsoft Dynamics 365 反欺诈 - 帐户保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线客户帐户。
- Microsoft Dynamics 365 反欺诈 - 丢失防护：使用此产品，零售商可以减少其实体商店的缩减。
- Microsoft Dynamics 365 反欺诈 - 购买保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线交易。
- Microsoft Dynamics 365 反欺诈 - 帐户保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线客户帐户。

停用 Microsoft Dynamics 365 反欺诈产品：

以下 Microsoft Dynamics 365 反欺诈产品/服务将于 2020 年 7 月 1 日从价目表中删除：

- Microsoft Dynamics 365 反欺诈
