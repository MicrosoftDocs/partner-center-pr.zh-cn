## <a name="billing-frequency-default-changing-from-annual-to-monthly"></a><a name="4"></a>计费频率默认从“按年”更改为“按月”

### <a name="categories"></a>类别

- 日期:2020-06-17
- 键入：功能
- 状态：即将推出

### <a name="summary"></a>摘要

在合作伙伴中心购买体验中，计费频率默认从“按年”更改为“按月”。

### <a name="impacted-audience"></a>影响受众

云解决方案提供商 (CSP) 直接提供商和间接提供商

### <a name="details"></a>详细信息

计费频率设置可确定针对合作伙伴已购买的服务向其计费的频率。 在合作伙伴中心目录中进行购买时，合作伙伴会在“审阅”步骤中看到此设置。 此设置默认为“按年”，因为已针对基于许可证的服务引入了按年计费。 根据合作伙伴的反馈，合作伙伴中心团队从 6 月 17 日起会将此默认值从按年更改为按月，以便更符合合作伙伴预期的情况 。 合作伙伴可以在审阅页上提交交易之前或[在从管理订阅页提交交易之后]( https://docs.microsoft.com/partner-center/billing-annual-monthly#change-billing-frequency)对此设置进行更改。 请注意，仅支持按年计费的产品/服务才可继续沿用默认的“按年”计费。 此默认值更改仅适用于合作伙伴中心门户的用户体验，合作伙伴中心 API 的工作方式不会更改。

我们希望这种更改能降低合作伙伴在无意中意外购买按年计费订阅的可能性。

## <a name="changes-to-microsoft-365-f1"></a><a name="3"></a>对 Microsoft 365 F1 的更改

### <a name="categories"></a>类别

- 日期:2020-06-15
- 键入：产品/市场
- 状态：现已推出

### <a name="summary"></a>摘要

将通过新服务更新 Microsoft 365 F1。 将于 6 月 15 日发布新产品/服务以替换之前的 Microsoft 365 F1。

### <a name="impacted-audience"></a>影响受众

云解决方案提供商 (CSP) 直接提供商和间接提供商

### <a name="details"></a>详细信息

自 2020 年 4 月起，合作伙伴中心已开始提供 Microsoft 365 F1。 将通过添加对新服务的支持来增强功能：Exchange Kiosk 和 MS 搜索功能。 到 6 月底，将根据原始 F1 产品/服务在所有现有订阅上自动预配新服务。 原始 F1 产品/服务将于 6 月 15 日替换为用于新订阅购买的新 F1 产品/服务（如下所示）。

原始 F1 产品/服务：

   |**产品/服务名称**|**产品/服务 ID**|**材料 ID**|**预配 ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|1ce4a4ba-65f7-49e7-bb28-d92d07d58b86|1PI-00005|M365_F1|
   |Microsoft 365 F1 试用版|dd97e36a-f0df-4a9a-b461-625420656511|1PI-00006|M365_F1|

希望购买新 F1 订阅的合作伙伴应在 6 月 15 日前使用以下产品/服务 ID。 合作伙伴可以在合作伙伴中心目录中查看这些 ID。 利用合作伙伴中心 API 的合作伙伴也可以使用以下信息。

新 F1 产品/服务：

   |**产品/服务名称**|**产品/服务 ID**|**材料 ID**|**预配 ID**|
   |-------------------|:------|:------|:------|
   |Microsoft 365 F1|17b4a1d6-2c65-478e-a046-51cf40d25d76|1PI-00012|M365_F1_COM|
   |Microsoft 365 F1 试用版|2e4e9424-06db-4d35-b2d8-3a1b0ef1dc63|1PI-00013|M365_F1_COMM|

### <a name="call-to-action"></a>行动号召
具有当前 Microsoft 365 F1 产品的合作伙伴无需进行任何操作。 Exchange Kiosk 和 MS 搜索的服务计划将在 6 月底基于现有服务计划进行回填。

希望在 6 月 15 日之后购买 Microsoft 365 F1 的合作伙伴应使用新的 F1 产品/服务信息，以确保所购买的是正确的产品/服务。 新的 F1 产品/服务 ID 和其他信息将在 7 月 1 日于云解决方案提供商价目表更新中更新。

## <a name="new-and-end-of-life-microsoft-dynamics-365-and-power-platform-offers-and-products"></a><a name="1"></a>新的和已停用的 Microsoft Dynamics 365 和 Power Platform 产品及服务

### <a name="categories"></a>类别

- 日期:2020-06-11
- 键入：产品/市场
- 状态：即将推出

### <a name="summary"></a>摘要

Microsoft 将于 2020 年 7 月 1 日发布新的 Dynamics 365 和 Power Platform 产品及服务，并停用相应的反欺诈产品。

### <a name="impacted-audience"></a>影响受众

全球范围内通过云解决方案提供商 (CSP) 计划提供 Dynamics 365 的合作伙伴（除非另有规定）

### <a name="details"></a>详细信息

新的 Dynamics 365 反欺诈产品：

- Microsoft Dynamics 365 反欺诈 - 帐户保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线客户帐户。
- Microsoft Dynamics 365 反欺诈 - 丢失防护：使用此产品，零售商可以减少其实体商店的缩减。
- Microsoft Dynamics 365 反欺诈 - 购买保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线交易。
- Microsoft Dynamics 365 反欺诈 - 帐户保护：此产品通过检测及阻止欺诈性活动帮助商家保护其在线客户帐户。

停用 Microsoft Dynamics 365 反欺诈产品：

以下 Microsoft Dynamics 365 反欺诈产品/服务将于 2020 年 7 月 1 日从价目表中删除：

- Microsoft Dynamics 365 反欺诈
- Microsoft Dynamics 365 反欺诈 Addl 评估
