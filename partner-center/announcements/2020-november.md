## <a name="new-microsoft-teams-advanced-communications-skus-for-government-community-cloud-gcc-coming-soon"></a><a name="4"></a>即将推出适用于政府社区云 (GCC) 的新 Microsoft Teams 高级通信 SKU

### <a name="categories"></a>类别

- 日期:2020-11-2
- 产品/市场

### <a name="summary"></a>摘要

GCC 中新 Microsoft Teams 高级通信加载项 SKU 将于 2020 年 11 月 1 日推出。

### <a name="impacted-audience"></a>影响受众

参与云解决方案提供商 (CSP) 计划的所有合作伙伴

### <a name="details"></a>详细信息

GCC 中新 Microsoft Teams 高级通信加载项的定价为每用户每月 12 美元。 可以在包含 Microsoft Teams 的任何其他 Microsoft 365 套件基础上购买该加载项 SKU。 高级通信提供一组新的用于大型会议、通信策略、集成和 IT 管理的高级工具的功能。 

#### <a name="offer-details"></a>产品/服务详细信息

   |**产品/服务名称**|**产品/服务 ID**|**材料 ID**|
   |-------------------|:------|:------|
   |适用于 GCC 的高级通信|56fe76f5-f4ba-4fac-9561-d0daf59b01a1|7FB-00003|

#### <a name="frequently-asked-questions"></a>常见问题 

**什么是高级通信？** 这种新的 Microsoft Teams 加载项使客户能够进一步增强其通信体验。 可以在具有相关订阅的任何 Microsoft 365 SKU 的基础上购买该加载项。

**它的费用是多少？** 商用 ERP 的定价是每用户每月 12 美元。

**哪些客户可以购买该加载项？** GCC 客户可以购买该加载项。

**如何购买它？** 可通过企业协议、企业协议订阅、教育解决方案合约、CSP 或 Web Direct 购买该加载项。

**它在何处销售？** 它可以在美国市场中销售。

**先决条件是什么？** 任何包含 Microsoft Teams 的 Microsoft 365 或 Office 365 套件均可安装该加载项。

与组织中的相关联系人共享此信息，并了解追加销售和交叉销售机会。 查看 [Teams 合作伙伴指南](https://aka.ms/teamscallingmeetingsguide)的“资源”部分。

## <a name="introducing-microsoft-365-business-voice-for-nonprofit-organizations"></a><a name="2"></a>为非营利性组织推出 Microsoft 365 商务语音

### <a name="categories"></a>类别

- 日期:2020-11-2
- 产品/市场

### <a name="summary"></a>摘要

2020 年 11 月 1 日，Microsoft 为非营利性组织推出了新的 Microsoft 365 商务语音 SKU。

### <a name="impacted-audience"></a>影响受众

直接提供商、间接提供商和间接经销商

### <a name="details"></a>详细信息

2020 年 11 月 1 日，Microsoft 推出了新的商务语音 SKU。 完整套件在加拿大、英国和美国提供。 商务语音（不含通话套餐）将在所有其他市场提供。 

Microsoft 365 商务语音是一种面向中小型企业的基于云的电话系统，内置于 Office 365 中。 将商务语音添加到客户的 Office 365 订阅中，便可在单个应用程序（即 Microsoft Teams）中提供一个集通话、聊天和会议于一体的通信和协作解决方案。

有关定价详细信息，请参阅价目表。

可将 Microsoft 365 商务语音添加到以下订阅，最多供 300 名用户使用：

- Office 365：商业协作版、商业高级版、A1、E1、A3 和 E3
- Microsoft 365：商业、A3 和 E3

- 产品/市场
