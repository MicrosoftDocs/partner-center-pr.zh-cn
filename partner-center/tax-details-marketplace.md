**相应的角色**：全局管理员

本页为商业市场发布者 (发布者) 和消费者市场开发人员 (开发人员) 分别与最终客户购买符合 [Microsoft Azure 市场](/legal/marketplace/msft-publisher-agreement) 发布者协议和 [应用开发人员协议](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE4o4bH)的发布者/开发人员产品有关的税务责任和税务注意事项。  以下信息是 Microsoft 发布者协议和协议条款中条款和条件的新增内容，而不是应用开发人员协议。  除非另有说明，否则以下信息适用于商业市场发布者和使用者市场开发人员，这些开发人员通过 Microsoft 市场销售其产品/产品/产品/服务。 

## <a name="end-customer-taxation"></a>最终客户税务

Microsoft 和发布者/开发人员对最终客户销售 (**最终** 客户税务) 的销售、使用、商品和服务、增值或类似税款（如果有）的责任取决于发布者/开发人员选择通过 Microsoft 市场 () 向最终客户销售产品/服务的国家/地区。 下表列出了在销售符合条件的国家/地区，最终客户税款的情况Microsoft 市场 () 。  以下信息可能会更改，发布者/开发人员负责查看此页面，了解当前的准确信息。

在 **发布者/开发人员管理的国家**/地区，Microsoft 充当发布者/开发人员 (代理或) 代理。 发布者/开发人员确认并同意，发布者/开发人员单独负责确定最终客户税务的所有合规性以及管理最终客户税务，包括但不限于注册、税务计算、税务收取、税务登记、验证客户的业务状态以及为客户提供税务发票。 发布者/开发人员确认，对于发布者/开发人员托管国家/地区的任何销售，Microsoft 当前可能无法提供发布者/开发人员为客户提供税务发票所需的交易详细信息。 

在 **Microsoft 托管的国家/地区**，Microsoft 充当发布者/开发人员的代理。 Microsoft 负责管理最终客户税款，包括计算、收集和/或发出某些税款。 对于商业市场，这可能包括通过获取税务注册号或豁免证书来验证最终客户的业务状态。 Microsoft 通过本地关联公司或远程关联公司（在本地注册）管理最终客户税务。 Microsoft 的注册号下的 Microsoft 发票。 Microsoft 包括从最终客户收取的 Microsoft 税务返回单上的任何税款。

Microsoft 不保证 Microsoft 的操作将完全履行 Microsoft 托管国家/地区中的发布者/开发人员义务。  对于所有 Microsoft 管理的国家/地区，Microsoft 强烈建议发布者/开发人员与其自己的税务顾问合作，以确定 Microsoft 管理的税务监管是否足以满足发布者/开发人员的合规性要求。 对于发布者/开发人员销售其产品/服务的任何 Microsoft 托管国家/地区，这一点尤其重要。  Microsoft 充当 Microsoft 托管国家/地区发布者/开发人员的代理。

在 **经销商国家**/地区，Microsoft 充当经销商，而不是发布者/开发人员的代理，通过发布者/开发人员产品/服务通过 Microsoft 市场 (销售) 。  发布者/开发人员负责向 Microsoft 销售产品/服务的销售、使用、商品和服务、增值或类似税款（如果有）。Microsoft 负责向最终客户转售的所有最终客户税款。

## <a name="tax-considerations-on-payouts"></a>付款的税务注意事项

在发布者/开发人员管理的国家/地区，Microsoft 向发布者/开发人员支付的费用不包括商店费用以及任何适用的税款，其中可能包括以下任一项：预扣税和销售、使用、商品和服务、增值或商店费用的类似税款。

在适用的 Microsoft 托管国家/地区以及经销商中，Microsoft 向发布者/开发人员支付的费用不包括商店费用以及任何适用的税款，其中可能包括以下任一项：最终客户税、预扣税和销售、使用、商品和服务、增值或商店费用的类似税款。 Microsoft [](payout-statement.md)提供给发布者/开发人员的报告标识了这些税款金额。 

Microsoft 针对以下国家/地区中的商店费用收取税款，其中发布者/开发人员选择通过以下国家/地区向最终客户Microsoft 市场 (产品/) ：

|**国家/地区**|**商业市场**|**消费者市场**|
|----------------|-----------------------------|-----------------------|
|澳大利亚|服务费用税|服务费用税|
|Canada|服务费用税|服务费用税|
|墨西哥|空值|服务费用税|
|新西兰|服务费用税|服务费用税|
