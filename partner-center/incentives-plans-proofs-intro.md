# <a name="use-these-resources-to-help-you-create-incentives-plans-and-proofs-of-execution"></a>使用这些资源来帮助你创建奖励计划和执行证明

**适当角色**：奖励管理员|奖励用户

本文介绍成功并获取特定奖励计划的工作付款所需的一些关键文档。 这些文档包括：

- 奖励合作申请计划
- PoE (的详细) 

根据你注册的奖励计划的类型，奖励合作申请计划和执行证明 (PoE) 将有助于你成功获得奖励。

## <a name="what-is-an-incentives-co-op-claims-plan"></a>什么是奖励合作申请计划？

奖励合作申请计划描述营销活动、活动以及使计划成功所需的其他详细信息。 创建此计划后，需要将其提交到 Microsoft 进行审批。 详细了解如何 [创建计划](incentives-create-your-plan.md)。

## <a name="what-is-a-proof-of-execution-poe"></a>什么是 PoE (的执行) ？

获得奖励工作的付款后，必须将 PoE (的详细执行) Microsoft。 PoE 可能包含有关任何相关市场活动、奖励活动发生日期和认领金额的详细信息。 

详细了解如何 [准备执行证明](incentives-prepare-your-proof-of-execution.md)。
