# <a name="core-requirements-for-the-activities-in-your-incentive-program"></a>关于奖励计划中的活动的核心要求

**相应的角色**

- 奖励管理员
- 奖励用户

要针对其提交奖励声明的每种类型活动都具有核心要求。 你需要在执行文档的证明中说明你已满足这些要求。 某些活动还需要特定的通信类型和指标。

下表概述了每种可接受活动的核心要求。 

## <a name="demand-generation"></a>需求生成

### <a name="print-advertising"></a>印刷广告

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求    | 公司品牌和联系人信息<br><br>Microsoft 内容 |
| 通信要求 | 最终通信的副本（基于广告）显示计划特定的核心要求 |
| 指标     | 估计的客户（或合作伙伴）覆盖范围 |
|符合条件的支出    | 第三方媒体投放<br><br>印刷材料<br><br>费用 - 代理营销和创意服务，包括代理费  |

### <a name="digital-advertising"></a>数字广告

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求 | 不可用 |
| 通信要求 | 数字广告的屏幕截图<br><br>AND<br><br>登陆页面 URL，显示计划特定的核心要求 |
| 指标     | 结果摘要（例如单击浏览式报告） |
|符合条件的支出    | 第三方媒体投放<br><br>费用 - 代理：营销和创意服务，包括代理费 |

### <a name="direct-mail-email-and-short-message-service-sms"></a>直邮邮件、电子邮件和短信服务 (SMS)

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求    | 不可用 |
| 通信要求 | 1) 最终通信的副本显示计划特定的核心要求<br><br>2) 分发证明可以包括带邮戳的运输信封或其他邮寄证明（直邮邮件）、显示带日期标头或其他电子邮件证明的最终电子邮件，或短信 (SMS) 的照片 |
| 指标     | 估计的客户（或合作伙伴）覆盖范围 |
|符合条件的支出    | 数据库收购<br><br>印刷材料<br><br>邮资<br><br>费用 - 代理：营销和创意服务，包括代理费 |

### <a name="partner-website-and-search-engine"></a>合作伙伴网站和搜索引擎

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求    | 不可用 |
| 通信要求 | 网页 URL 和/或显示 URL 的屏幕截图<br><br>SEO 项目的工作声明或服务协议（仅限 SEO 活动） |
| 指标     | 结果摘要（例如单击浏览式报告） |
|符合条件的支出    | 费用 - 代理：营销服务，包括代理费<br><br>编程/开发：编程和开发费<br><br>费用 - SEO：优化和维护费  |

### <a name="microsoft-syndicated-content"></a>Microsoft 综合内容

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求    | 不可用 |
| 通信要求 | 网页 URL 和/或显示 URL 的屏幕截图 |
| 指标     | 结果摘要（例如单击浏览式报告） |
|符合条件的支出    | 每个使用周期的实际成本高达 18,750 美元，包括：费用 - 代理营销服务（包括代理费）<br><br>编程和开发费<br><br>媒体投放  |

### <a name="multi-touch-digital-campaign"></a>多点触控的数字市场活动

| 类别 | 执行证明    |
| ------ | ----------- |
| 核心要求    |不可用 |
| 通信要求 | 需要提供适用活动页面中详述的所有执行证明<br><br>数字广告<br><br>直接电子邮件<br><br>合作伙伴网站和搜索引擎优化<br><br>电话营销  |
| 指标     | 不可用 |
|符合条件的支出    | 第三方媒体投放<br><br>费用 - 代理：营销和创意服务，包括代理费<br><br>数据库收购<br><br>编程/开发：编程和开发费<br><br>费用 - SEO：优化和维护费<br><br>费用 - 第三方呼叫中心费用 |

### <a name="internal-incentives-and-spiffs"></a>内部奖励和 SPIFF

| 类别 | 执行证明    |
| ------ | ----------- |

| ------ | ----------- |
