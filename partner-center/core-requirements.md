# <a name="core-requirements-for-incentives-co-op-claims"></a>奖励合作申请的核心要求

**适用于**

- 合作伙伴中心

**相应的角色**

- 奖励管理员
- 奖励用户

提交奖励申请的每种类型的活动都有需要满足的核心要求，需要在执行证明文档中展示这些要求的符合情况。 在某些情况下，还需要一些已定义的通信类型和指标。

下表概述了每种可接受活动的核心要求。
