# <a name="bulk-operations-for-co-sell-opportunities-using-comma-separated-value-csv-files"></a>使用逗号分隔值和 CSV 文件对联合销售 (批量) 操作

**适当的角色**：引荐管理员|引荐用户

批量运营合作伙伴中心有助于公司导出和导入联合销售机会数据。 导航到 **联合销售机会** 页，在页面标题横幅的右上方找到导入和导出链接。 同时具有 **引荐管理员和****引荐用户** 权限的用户可以使用此功能。

> [!IMPORTANT]
> 通过批量导入完成创建/更新操作是不可逆的。 修改或创建大量记录时请小心。 创建交易后，只能修改字段的子集。 **一旦任何交易达到终端状态（如"已拒绝/已过期/失败/丢失"，将不允许任何操作。**

## <a name="export-co-sell-opportunities"></a>导出联合销售机会

以下信息介绍了导出功能：

- 单击"导出"按钮最多可以导出 **5000** **条** 记录。
- 下载的交易将基于你的访问级别。 引荐管理员和引荐用户可能会根据他们的范围和作为团队成员包含在交易中而获得不同的结果。 详细了解 [引荐权限](permissions-overview.md#manage-referrals)。
- 导出函数将考虑联合销售机会页中的当前选项卡以及已应用的筛选器。
- 将生成一个 CSV 文件，该文件包含基于所应用筛选器的所有数据。
- 下载记录最多可能需要一分钟。
- 你不必等待下载操作完成。 即使导航到合作伙伴中心的其他页面，该文件也会在导出功能完成后立即下载。
- 您可以重复使用已下载的文件来修改交易详细信息并上传以更新任何记录。

## <a name="import-co-sell-opportunities"></a>导入共同销售机会

- 您可以使用导入功能创建或更新 **最多1000个记录** 。
- 可以通过从合作伙伴中心的 "导入" 页下载模板从头开始生成模板。
- 你还可以使用导出功能下载现有记录并对其进行更新。
- 如果文件包含的记录超过1000个，则无法对其进行处理。
- 在处理该文件后，将在最后一个进程文件卡中显示一个摘要，其中包含已创建、更新和未处理的引用的数量。
- 您可以下载已处理记录的详细信息，修复任何错误，并上传相同的文件以创建或更新在上一次运行时失败的记录。 **在上传在上一次运行中失败的已更正记录之前，请从文件中删除所有成功记录。**
- 若要添加更多解决方案，请在 "解决方案 1" 旁边添加额外的列，并将列名用作解决方案 X，其中 X 表示交易中的解决方案编号。 例如，解决方案2，解决方案3。
- 最多可以向一个交易添加50个解决方案。
- 若要添加更多团队成员，请在 "团队成员 1" 旁边添加额外的列，并使用 "列名称" 作为 "团队成员 X"，其中 X 表示交易中的团队成员的数量。 例如，团队成员2，团队成员3。
- 最多可以向一个交易添加50个团队成员。

> [!NOTE]
> 不必等待处理完成。 处理完成后，最后处理的文件的详细信息将可供下载。 **如果要上载包含1000记录的文件，可能需要长达10分钟的时间。**
